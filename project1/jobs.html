Cloud Engineer

Reference No: COS01

Position Title: Junior Cloud Engineer

The Role: Join our cloud engineering team to assist with designing, implementing, and maintaining cloud-based infrastructure, with opportunities to grow and innovate in a supportive environment.

Salary Range: $70,000 - $90,000 AUD per annum

Reports To: Senior Cloud Engineer

Why Cloud Engineering is Relevant

With the rapid shift toward cloud infrastructure across industries, we are in need of skilled engineers to build and maintain our systems. This role offers a fantastic entry point for aspiring professionals looking to grow with evolving technologies.

Key Responsibilities

  • Assist in deploying and managing cloud services across AWS and Azure.
  • Support senior engineers with troubleshooting and optimization tasks.
  • Write scripts for cloud automation tasks and system monitoring.
  • Ensure compliance with security policies and documentation standards.
  • Participate in code reviews and team knowledge-sharing sessions.

Required Qualifications & Skills

Essential

  • Basic knowledge of AWS and/or Azure platforms.
  • Proficiency in a scripting language such as Python or Bash.
  • Understanding of IAM and cloud security fundamentals.
  • Strong analytical and communication skills.
  • 1+ year experience through coursework, internships, or personal projects.

Preferable

  • Experience with Infrastructure-as-Code tools like Terraform or CloudFormation.
  • Exposure to DevOps tools like Docker or Jenkins.
  • Azure Fundamentals or AWS Certified Cloud Practitioner certification.

Cloud Systems Administrator

Reference No: COS02

Position Title: Cloud Systems Administrator (Level 2)

The Role: Manage hybrid cloud systems across AWS and Azure, ensuring their security, performance, and reliability while supporting mission-critical enterprise operations.

Salary Range: $95,000 - $120,000 AUD per annum

Reports To: Cloud Infrastructure Manager

The Importance of Cloud Systems Administration

As our partners adopt hybrid and multi-cloud strategies, experienced administrators play a vital role in ensuring security, performance, and reliability. This role supports mission-critical operations and provides real-world experience in enterprise settings.

Key Responsibilities

  • Monitor, maintain, and optimize cloud infrastructure.
  • Provision and manage virtual machines, storage, and network resources.
  • Automate system updates and deployments using Infrastructure-as-Code.
  • Implement security configurations and perform risk assessments.
  • Provide Tier 2 support for cloud-related service issues.

Required Qualifications & Skills

Essential

  • 3+ years in cloud or systems administration roles.
  • Advanced knowledge of Microsoft Azure and AWS platforms.
  • Experience managing Windows Server environments in cloud setups.
  • Strong scripting skills and experience with Terraform or CloudFormation.
  • Solid understanding of IAM, VPNs, firewalls, and data protection best practices.

Preferable

  • Relevant certifications (e.g., AWS SysOps Administrator, Azure Administrator Associate).
  • Experience with Kubernetes or container-based deployments.
  • Familiarity with CI/CD pipelines and monitoring tools.
